# 专家数字人平台 - 快速启动指南

## 启动方式

### 方式一：桌面快捷脚本（推荐）
双击桌面上的 `启动专家平台.sh` 文件，即可一键启动项目并自动打开浏览器。

### 方式二：项目内启动脚本
1. 进入项目目录：`/Users/qc/Desktop/Trae/zhuanjia`
2. 双击运行 `start.sh` 文件

### 方式三：AppleScript启动器
双击桌面上的 `启动专家平台.scpt` 文件，将以Mac应用程序风格启动项目。

### 方式四：命令行启动
```bash
cd /Users/qc/Desktop/Trae/zhuanjia
npm run dev
```

## 访问地址
项目启动后，请在浏览器中访问：http://localhost:5173/

## 停止服务
在运行服务的终端窗口中，按 `Ctrl+C` 即可停止服务。

## 注意事项
- 首次运行时，脚本会自动检查并安装项目依赖
- 确保已安装Node.js和npm
- 如果遇到端口占用问题，可以修改项目配置中的端口号
