# 专家数字人平台 - Windows快速启动指南

## 启动方式

### 方式一：桌面快捷脚本（推荐）
双击桌面上的 `启动专家平台.bat` 文件，即可一键启动项目并自动打开浏览器。

### 方式二：项目内启动脚本
1. 进入项目目录：`Trae\zhuanjia`
2. 双击运行 `start.bat` 文件

### 方式三：命令行启动
```cmd
cd Trae\zhuanjia
npm run dev
```

## 访问地址
项目启动后，请在浏览器中访问：http://localhost:5173/

## 停止服务
在运行服务的命令行窗口中，按 `Ctrl+C` 即可停止服务。

## 注意事项
- 首次运行时，脚本会自动检查并安装项目依赖
- 确保已安装Node.js和npm
- 如果遇到端口占用问题，可以修改项目配置中的端口号
- 在Windows系统中，建议使用Git Bash或WSL来获得更好的兼容性

## 创建桌面快捷方式（可选）
1. 右键点击桌面，选择"新建" > "快捷方式"
2. 在位置栏中输入：`cmd /c "cd /d %USERPROFILE%\Desktop\Trae\zhuanjia && start.bat"`
3. 点击"下一步"，为快捷方式命名（如"专家数字人平台"）
4. 点击"完成"创建快捷方式
5. 可以右键点击快捷方式，选择"属性" > "更改图标"来设置一个合适的图标
