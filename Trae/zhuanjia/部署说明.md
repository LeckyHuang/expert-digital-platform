# 外部专家数字人平台 - 部署说明

## 项目概述

外部专家数字人平台是一个连接专家与用户的数字平台，提供专家展示、搜索、详情查看以及管理后台等功能。

## 技术栈

- 前端框架：React 18.3.1
- 语言：TypeScript
- 构建工具：Vite 6.3.5
- 路由：React Router DOM 7.9.6
- 状态管理：Zustand 5.0.3
- 样式：Tailwind CSS
- 部署平台：Vercel

## 本地运行

如果需要在本地运行项目：

### 环境要求

- Node.js (推荐 v18 或更高版本)
- npm 或 yarn

### 安装依赖

```bash
cd /path/to/project
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 http://localhost:5173 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 部署更新

当需要更新部署时，只需将代码推送到GitHub的main分支，Vercel会自动触发重新部署：

```bash
git add .
git commit -m "更新内容描述"
git push origin main
```

## 注意事项

2. 如果遇到构建错误，可以在Vercel控制台查看详细日志
3. 可以在Vercel控制台配置自定义域名和环境变量
